# 校园失物招领系统 - 项目完成总结

## 📋 项目概述

本项目是一个面向校园场景的失物招领系统，支持**微信小程序**和**PC 端网页**双平台访问，实现了失物发布、招领匹配、认领申请、审核管理、消息通知等核心功能。

## 🚀 快速启动

### 1. 启动后端
```bash
cd backend
npm install
npm run dev
# 运行在 http://localhost:3000
```

### 2. 启动小程序
1. 打开微信开发者工具
2. 导入 `miniprogram` 目录
3. 关闭"校验合法域名"
4. 编译运行

### 3. 启动 PC 端
```bash
cd web-frontend
npm install
npm run dev
# 运行在 http://localhost:8080
```

### 后端环境变量 (.env)
```bash
# 服务器配置
PORT=3000
JWT_SECRET=your_secret

# 校园认证（可选）
CAMPUS_AUTH_URL=https://auth.xxx.edu.cn/api/verify
CAMPUS_AUTH_APP_KEY=your_key
CAMPUS_AUTH_APP_SECRET=your_secret

# 微信推送（可选）
WECHAT_APP_ID=wx123456
WECHAT_APP_SECRET=your_secret
WECHAT_SUBSCRIBE_TEMPLATE_ID=template_id

# 企业微信（可选）
WECHAT_CORP_ID=corp_id
WECHAT_CORP_SECRET=corp_secret
WECHAT_AGENT_ID=1000001
```
